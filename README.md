# COS202 — Fri-to-Mon Assignments (01/07/2026)

## 1. Mathematical Calculator (MC) — `calculator.py`
A simple, screen-based GUI calculator built with Python's Tkinter.

**Operations supported:** `+`  `-`  `*`  `/`  `\` (integer division)  `^` (power)  `%`  `C` (Clear)  `OFF` (Exit)

**Run it:**
```
python3 calculator.py
```

## 2. Personal Pocket CGPA Calculator (PPC) — `cgpa_calculator.py`
An interactive console program that computes a student's CGPA using
selection control statements (`if` / `elif` / `else`). Enter each
course's credit unit and score; the program grades it, then prints a
full result summary with total quality points, CGPA, and degree
classification.

**Run it:**
```
python3 cgpa_calculator.py
```
